A full-stack food pre-ordering platform built for a college mela that allows users to browse menus, place orders, and verify orders using OTP authentication. The application uses FastAPI for the backend, React for the frontend, and Google Sheets as a lightweight database. The platform handled 300+ users and generated 800+ page views during the event.

**Key Features**
• FastAPI backend with REST APIs  
• React frontend for dynamic user interface  
• OTP-based order verification  
• Google Sheets used as a lightweight database  
• Real-time order tracking and confirmation

**Tech Stack**
FastAPI, React, Python, JavaScript, Google Sheets API
